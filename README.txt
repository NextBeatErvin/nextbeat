NEXTBEAT NEON SZERKESZTHETŐ WEBOLDAL

Megnyitás:
1. Csomagold ki a ZIP fájlt.
2. Nyisd meg az index.html fájlt.

Oldalak:
- index.html = főoldal + regisztráció
- scanner.html = QR beolvasó
- admin.html = vendéglista
- style.css = kinézet
- config.js = rendezvény és Supabase beállítás
- app.js = regisztráció + QR generálás
- scanner.js = QR ellenőrzés
- admin.js = admin lista + CSV export
- setup.sql = Supabase adatbázis létrehozás

A bal felső logó minden oldalon kattintható, és visszavisz a főoldalra.

Szerkesztés:
- Szövegek: index.html
- Színek, méretek: style.css
- Rendezvény neve, dátuma: config.js
- Logók: assets mappa

Ingyenes próba:
- Supabase nélkül helyi próba módban működik.
- Több telefonos próbához Supabase kell.
